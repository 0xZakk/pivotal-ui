Collapse Toggle

Also see the select tabs for a toggle triggered by a select group. This is frequently used in combination with our panel components. NOTE: The 'alligators' (the > in the HAML below) are structural -- without them, the browser renders the whitespace on the anchor tags, and the images jump around by a few pixels when they're toggled.

Simple Example:

Expand me
Collapse me
Look at this content. 
%a.collapsed{data: {toggle: 'collapse'}, href: '#simple-collapse-example'}
  .when-collapsed
    Expand me
  .when-expanded
    Collapse me
#simple-collapse-example.collapse
  Look at this content.

Hoverable

This component is for showing hidden actions on hover. If you put this on an element then hovering on that parent element will show the hidden element.

Angular Sortable Table

USE:

  1. Add a sorter in your controller scope that has a reference to the collection you want to sort.
  2. Add "sortable" to any header/element which should trigger a sort (if the sortable directive doesn't do what you want, write a new one or just use ng-click. Please don't add more behavior to the current sortable).

Clickable

Use this style of table when rows are clickable. It is possible to prevent a user from clicking on a row w/ a message by adding data-prevent-click and data-prevent-click-message to the row. (N.B.: This does not actually add any sort of permissions to the URL in question). We are making the assumption here that table-hover will only be used if the table is clickable, because we add a pointer.

Style Clickability
Forbidden Links High
Buttons Medium
Typography Low 
%table.table.table-data.table-hover{:'data-behavior' => 'ClickableTable'}
  %thead
    %tr
      %th
        Style
      %th{colspan: 2}
        Clickability
  %tbody
    %tr{:'data-target-path' => '/style_guide#link', :'data-prevent-click' => 'true', :'data-prevent-click-message' => "YOU'VE ENTERED THE FORBIDDEN ZONE"}

      %td
        High
