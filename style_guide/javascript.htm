Collapse Toggle

Also see the select tabs for a toggle triggered by a select group. This is frequently used in combination with our panel components. NOTE: The 'alligators' (the > in the HAML below) are structural -- without them, the browser renders the whitespace on the anchor tags, and the images jump around by a few pixels when they're toggled.

My fancy application

This content should hide and show when the trigger is clicked. 
<div class='panel' id='accordion-1'>
  <div class='panel-heading collapse-trigger collapsed' data-parent='#accordion-1' data-toggle='collapse' href='#myCollapse-1'>
    <div class='row'>
      <div class='col-md-23'><a class='icon-when-collapsed'><i class='fa fa-plus-square'></i></a><a class='icon-when-expanded'><i class='fa fa-minus-square'></i></a><h3 class='collapse-title type-em-1 type-primary-2'>My fancy application</h3></div>
    </div>
  </div>
  <div class='panel-collapse collapse' id='myCollapse-1'>
    <div class='panel-body'>
      This content should hide and show when the trigger is clicked.
    </div>
  </div>
</div>

#accordion-1.panel
  .panel-heading.collapse-trigger.collapsed{data: {toggle: 'collapse', parent: '#accordion-1'}, href: '#myCollapse-1'}
    .row
      .col-md-23
        %a.icon-when-collapsed>
          %i.fa.fa-plus-square>
        %a.icon-when-expanded>
          %i.fa.fa-minus-square>
        %h3.collapse-title.type-em-1.type-primary-2> My fancy application
  #myCollapse-1.panel-collapse.collapse
    .panel-body
      This content should hide and show when the trigger is clicked.

Hoverable

This component is for showing hidden actions on hover. If you put this on an element then hovering on that parent element will show the hidden element.

List example:

This can be used with any kind of list but the list group style seems to fit this use case best.

  • List Item 1 Edit
  • List Item 2 Edit
  • List Item 3 Edit
  • Not hoverable 
<ul class='list-group' hoverable-group='true'>
  <li class='list-group-item' hoverable='true'>
    List Item 1

  </li>
  <li class='list-group-item' hoverable='true'>

  <li class='list-group-item' hoverable='true'>
    List Item 3

JS Tables

Angular Sortable Table

USE:

  1. Add a sorter in your controller scope that has a reference to the collection you want to sort.
  2. Add "sortable" to any header/element which should trigger a sort (if the sortable directive doesn't do what you want, write a new one or just use ng-click. Please don't add more behavior to the current sortable).
Name Total Apps Running Apps Stopped Apps Down Apps Service Instances Quota
{{tableItem.name}} {{tableItem.totalApps}} {{tableItem.runningApps}} {{tableItem.stoppedApps}} {{tableItem.downApps}} {{tableItem.serviceInstances}} {{tableItem.quota}} 
<table class='table table-data table-sortable' ng-cloak ng-controller='ExampleSpaceDataTableCtrl'>
  <thead>
    <tr>
      <th class='txt-m' sortable='name'>Name</th>
      <th sortable='totalApps'>Total Apps</th>
      <th sortable='runningApps'>Running Apps</th>
      <th sortable='stoppedApps'>Stopped Apps</th>
      <th sortable='downApps'>Down Apps</th>
      <th class='txt-m' sortable='serviceInstances'>Service Instances</th>
      <th class='txt-m' sortable='quota'>Quota</th>
    </tr>
  </thead>
  <tbody>
    <tr ng-repeat='tableItem in tableItems'>
      <td>{{tableItem.name}}</td>
      <td>{{tableItem.totalApps}}</td>
      <td>{{tableItem.runningApps}}</td>
      <td>{{tableItem.stoppedApps}}</td>
      <td>{{tableItem.downApps}}</td>
      <td>{{tableItem.serviceInstances}}</td>
      <td>{{tableItem.quota}}</td>
    </tr>

</table>

Clickable

Use this style of table when rows are clickable. It is possible to prevent a user from clicking on a row w/ a message by adding data-prevent-click and data-prevent-click-message to the row. (N.B.: This does not actually add any sort of permissions to the URL in question). We are making the assumption here that table-hover will only be used if the table is clickable, because we add a pointer.

Style Clickability
Forbidden Links High
Buttons Medium
Typography Low 
<table class='table table-data table-hover' data-behavior='ClickableTable'>
  <thead>
    <tr>
      <th>
        Style
      </th>

        Clickability
      </th>
    </tr>
  </thead>
  <tbody>
    <tr data-prevent-click-message="YOU'VE ENTERED THE FORBIDDEN ZONE" data-prevent-click='true' data-target-path='/style_guide#link'>

        Forbidden Links
      </td>
      <td>
        High
      </td>
      <td class='txt-m txt-r'>

    <tr data-target-path='/style_guide#button'>

        Buttons
